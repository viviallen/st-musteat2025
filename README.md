# st-musteat2025

### 2025 必吃榜/年度总结生成器 <br>
*年底了酒馆人也要有自己的年度总结*
##### -> https://st-musteat2025.netlify.app/
<br>

## 📌 核心功能：
- 自填角色名称、作者（可填）、爱吃榜星级、标签（可自己另外加）、评论等信息，自带猫系/狗系/猫狗二象性分类标签
- 头像：可上传，也支持自动生成默认固定emoji头像
- 可勾选多人卡，可标注内含角色
- 双主题切换：日夜模式切换
- 预览：实时预览报告及返回修改，支持编辑与删除，操作流畅
- 一键导出： 生成长图报告+导出Excel表格
